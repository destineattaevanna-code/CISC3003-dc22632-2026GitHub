Demo 01 — Responsive One‑Page Layout

A semantic, mobile‑first page demonstrating a hero section, cards, gallery, and consistent components.

See Result Tech Stack

Overview

Goals

  • Apply semantic HTML for structure and accessibility.
  • Build responsive sections with a shared design system.
  • Keep styles clean and maintainable.

Features

  • Sticky header & skip‑link for a11y.
  • Reusable buttons (primary / ghost / small).
  • Responsive cards and gallery layout.

Tech Stack

  • HTML5 + semantic landmarks
  • CSS custom properties, Flexbox/Grid
  • No JS required

Accessibility Checklist

Basics

  • Language set on <html lang="en">
  • Descriptive alt text on images
  • Logical heading order

Navigation

  • Skip‑to‑content link
  • Keyboard‑navigable menu
  • Visible focus via browser default

Color

  • High‑contrast text on dark background
  • Links visually distinct

Result

The final layout renders properly from mobile to desktop with consistent spacing and card rhythm.
